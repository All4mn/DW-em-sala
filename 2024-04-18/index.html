Code

Elas são usadas para estruturar o conteúdo de um documento, para formatar o texto, para adicionar imagens e outros elementos, etc.

Samp

Destina-se a marcar textos que são mostrados como exemplo de saída de um programa.

Pre

é a tag utilizada para representar texto pré-formatado.

Exemplo 

            
                print("Hello World")
                x = 2
                if x > 0:
                    print("Maior que zero")

A saída do código será 

        
            Hello World 
            Maior que zero
